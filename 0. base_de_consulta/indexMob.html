Como Funciona

Configure sua apólice com base em dados objetivos como:

  • Precipitação pluviométrica (mm)
  • Temperaturas máximas/mínimas
  • Índices de umidade do solo

2. Contrato Automatizado

Seu seguro é codificado em um smart contract na blockchain com:

  • Termos imutáveis e transparentes
  • Execução automática quando condições são atendidas
  • Segurança criptográfica de nível bancário
  • Auditoria pública de todas as transações

Nenhuma parte pode alterar os termos ou impedir pagamentos devidos.

// Smart Contract - Seguro Paramétrico
contract AgriculturalInsurance {
  address public farmer;
  uint public coverageAmount;
  uint public rainfallThreshold;
  bool public paymentTriggered = false;

  constructor(address _farmer, uint _amount, uint _threshold) {
    farmer = _farmer;
    coverageAmount = _amount;
    rainfallThreshold = _threshold;
  }

  function checkAndPay(uint _actualRainfall) external {
    require(!_paymentTriggered, "Payment already made");
    if (_actualRainfall < rainfallThreshold) {
      payable(farmer).transfer(coverageAmount);
      paymentTriggered = true;
    }
  }
}
